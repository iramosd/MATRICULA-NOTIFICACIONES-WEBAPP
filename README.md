## 📋 Requisitos del sistema

- PHP 8.2+
- Laravel v12
- Node.js v22
- MySQL 8.0+
- Composer 2.5+

## 🚀 Instalación

### Método recomendado: Docker (Más fácil)

```bash
# Desde la raíz del proyecto:
docker compose up --build -d
```bash

El sistema estará disponible en:
👉 http://localhost:8000
